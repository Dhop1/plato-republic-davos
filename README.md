# Plato Republic / DavOS

- **App name:** Plato Republic / DavOS
- **Stack:** Node.js (Express) + Python (Flask)
- **Required environment variables:**
  - `PORT` (set by Railway automatically)
  - `FLASK_PORT` (default 5001)
  - `DATABASE_URL`
  - `SESSION_SECRET`
  - `GEMINI_API_KEY`
  - `NODE_ENV=production`
- **Deploy pipeline:** Cursor → GitHub → Railway
- **Start command:** `bash start.sh`
